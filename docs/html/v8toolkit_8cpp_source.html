223 /** Attempts to load the specified external resource.
224 * Attempts to load exact match filename in each path, then <filename>.js in each path, then <filename>.json in each path
225 *
226 * Paths are attempted in vector order[0, 1, 2...]. If the matching filename ends in .js (either because it was specified as such or because
227 * it matched after the suffix was added) it will be executed as a traditional module with the exports object being returned. If the matching
228 * filename ends in .json, the last value in the file will be returned.
229 * The results of require'd files is cached and if the same module (based on the full matching value including path and any added suffix) is
230 * required again, the cached value will be returned. The module will not be re-run.
231 *
232 * The goal of this function is to be as close to node.js require as possible, so patches or descriptions of how it differs are appreciated.
233 * Not that much time was spent trying to determine the exact behavior, so there are likely significant differences
234 */
